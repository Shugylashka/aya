Java сабағына қош келдіңіз!



Java — Sun Microsystems компаниясының жасап шығарған объектіге-бағытталған бағдарламалау тілі. Java қосымшалары әдетте арнайы байт-кодта компиляцияланады, сол себепті олар кез келген виртуалды Java-машинасында (JVM) компьютерлік архитектурасына тәуесіз орындалады. Ресми шығу күні — 23 мамыр 1995 жыл. Әзірлеуді Java қауымдастық процесі арқылы ұйымдастырылған қауымдастық жүргізеді, тілі және оны жүзеге асыратын негізгі технологиялар GPL лицензиясы бойынша таратылады.
